Agent Based Model

Programming for Geographical Information Analysis

Programming an Agent Based Model in Python
for the Programming for Geographical Information Analysis module.

The model:

- Builds agents in a space.

- Gets them to interact with eachother.

- Reads in environmental data.

- Gets agents to interact with the environment.

- Displays the model as an animation.

- Is contained within a Graphical User Interface.

- Is initialised with data from the web.
